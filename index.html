Quick Guides & Settings

Starting your shop is incredibly simple. Follow these steps or customize your account.

Guides

  • Create Bot: Go to @BotFather -> /newbot -> Name -> Copy Token -> Paste in Main Bot Settings.
  • Add Product: Admin Panel 2 -> Add Product -> Select Category -> Send Name -> Send Price -> Description -> Media -> Confirm.
  • Setup Automation: Create command -> Add content -> Link command to button in Manage Automation.

App Settings

  • 🔔 Notification: On/Off Noti.
  • 👤 Edit My Profile: Edit buyer info (name, phone, address).
  • 🤖 My Bots: See all your active bots.
  • 📜 T&C: Terms and Conditions of the bot.
  • 📈 Statistics: Shop stats (revenue, growth, top sales). Can extract to Excel.
  • ⭐️ My Plan: View your current bot plan.
